* サンプル.avif
* readme.md



### 使い方

1. サンプル記録シート3種を全てGoogleドライブにアップロード(フォルダはどこでも良い)
2. アップロードしたファイルを全てGoogleスプレッドシートとして保存 (Excelの形式のままだと、外部アプリケーション経由では編集できない)
3. Googleスプレッドシートに変換したファイルのシートIDを確認して「サンプル試合情報ファイル\_(注)シートIDはダミー.json」のシートIDを書き換える
   または試合情報ファイル生成ツール(https://ko-wgaca.f5.si/scoreboard/Match-Info-Generator/index.html)にて生成する。
   生成するときの入力例は「サンプル.avif」を参照。
4. 実際に使ってみる(https://ko-wgaca.f5.si/scoreboard/)
